(1) INTRODUCTION
-----------------
The Distributed Class Management System provides an infrastructure for teachers and student records which are shared across three data centres using Java RMI,multithreading and UDP Socket programming.

(2) REQUIRED SYSTEMS
--------------------
-Windows 10 
-Eclipse IDE

(3) HOW TO RUN APPLICATION?
-----------------------------
1. Import the application into the workspace
2. Setup the RMI environment
3. Execute three Server files are 
	a. MTLServer.java
	b. DDOServer.java
	c. LVLServer.java
4. Client manual testing file is : SingleClientTest.java 
5. Automated Test cases execute : MultiClientTest.java
